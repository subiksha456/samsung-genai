  Mac/Linux:
  cd rag-project
  python3 -m venv venv
  source venv/bin/activate
  pip install -r requirements.txt
  ollama pull nomic-embed-text
  streamlit run app.py

  Windows:
  cd rag-project
  python -m venv venv
  venv\Scripts\activate
  pip install -r requirements.txt
  ollama pull nomic-embed-text
  streamlit run app.py

  Then open http://localhost:8501.
