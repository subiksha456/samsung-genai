Investigation 3 — Bias Swap Test
Bias · Same qualifications, only the name changes
▾
This is a controlled experiment, not a single prompt. You will run the exact same profile through the exact same model twice — identical qualifications, only the first name changes. Why Your Results May Differ From Your Classmate's

Even with the same prompt and same model version, two students can get different outputs. Here is why — and why it matters for your career.

🎲
Temperature — Built-in Randomness
Every LLM has a temperature setting that controls how much randomness is in each response. The default is not zero. So the same prompt, same model, same second — can produce different words. This is by design, not a bug.
🧠
Different Training, Different Personality
ChatGPT, Claude, and Gemini were each trained on different datasets with different human feedback (RLHF). This gives each model a distinct "personality" — same question, different character, different failure patterns.
🗂️
Your Personal Context
ChatGPT's Memory feature stores facts about you across conversations. If two students have different stored context, the same prompt gives different answers. This is why disabling memory (Step 2 above) matters for a fair test.
Enterprise takeaway: This is exactly why production AI systems use API endpoints, not personal chat accounts — engineers set temperature = 0 for reproducible outputs, pass a controlled system prompt every time, and guarantee zero personal memory. What you did today is the uncontrolled version. And it's also why bias audits like Investigation 3 have to be run repeatedly, not just once — a single run can look clean by chance even when the underlying tendency is real.
